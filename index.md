# About

KC868-Uair Magic Box  

hardware resource:  

433MHz RF transmit  
433MHz RF receiver  
IR sender  
IR receiver  
Bottom-RGB 4 LEDs  
Vertical-RGB 1 LED  
2 channel analog input (4-20mA)  
2 channel analog output (DC 0-10V)  
DS18B20 temperature sensor  
extend port for any IIC sensor  
Buzzer
